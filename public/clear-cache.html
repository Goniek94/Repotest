Informacje o cache

To narzędzie pozwala wyczyścić dane przechowywane w przeglądarce, które mogą powodować problemy z wyświetlaniem marek samochodów w wyszukiwarce.

Ładowanie informacji o cache...

Uwaga: Po wyczyszczeniu cache, należy odświeżyć stronę główną, aby pobrać nowe dane z serwera.

Rozwiązywanie problemów

Jeśli po wyczyszczeniu cache nadal występują problemy z wyszukiwarką:

  1. Upewnij się, że serwer backend jest uruchomiony
  2. Sprawdź, czy API zwraca poprawne dane (endpoint: /api/car-brands/all-data)
  3. Wyczyść całą pamięć podręczną przeglądarki (Ctrl+F5 lub Cmd+Shift+R)
  4. Spróbuj otworzyć stronę w trybie incognito
